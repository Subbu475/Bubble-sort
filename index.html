What is Bubble Sort?

Bubble Sort is a simple sorting algorithm that works by repeatedly stepping through the list, comparing adjacent elements, and swapping them if they are in the wrong order. This process is repeated until no more swaps are needed, meaning the array is sorted.

How Bubble Sort Works

Bubble Sort compares pairs of adjacent elements, starting from the beginning of the array. If the first element is greater than the second, they are swapped. This process is repeated for each pair in the array. After each pass through the array, the largest element "bubbles" up to its correct position at the end.

Step-by-Step Process:

  • Step 1: Compare the first two adjacent elements of the array.
  • Step 2: Swap them if the first element is greater than the second.
  • Step 3: Move to the next pair of adjacent elements and repeat.
  • Step 4: Continue the process until the entire array is sorted.
  • Step 5: After each pass, the largest unsorted element is guaranteed to be in its correct final position.

  • It is an O(nÂ²) algorithm, meaning its performance degrades quickly as the size of the array increases.
  • It is simple and easy to understand, but inefficient for large datasets.
